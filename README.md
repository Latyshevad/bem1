## Шаблон Grunt-проекта для верстки

Для сборки требуется:
1. `nodejs` версии не ниже 6.9
1. `grunt` весрии не ниже 1.2
    > `npm install grunt -g`
    
Установка зависимостей для сборки:
> `npm install`

Сборка проекта:
> `grunt`

Сборка проекта c минифицированными скриптами и стилями:
> `grunt prod`

Вотчер:
> `grunt watch`
